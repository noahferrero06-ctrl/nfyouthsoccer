# NF Youth Soccer Website

A simple, modern, mobile-friendly website for NF Youth Soccer.

## Files

- `index.html` — main page
- `styles.css` — styles

## Run locally

1. Open `index.html` directly in your browser,
   or serve via local server:

   - Python 3: `python3 -m http.server 8000`
   - Then open `http://localhost:8000`

2. Update text, images, and contact details in `index.html`.
